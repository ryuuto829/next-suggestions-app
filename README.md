## Getting Started

1. Install dependencies

```bash
yarn
```

2. Change environment variables. See `.env.example` for an example.

3. Run the development server

```bash
yarn dev
```

## Built Using

- Next.js
- Vercel
- Firebase
- Tailwind CSS
- Typescript
- React Hook Form
